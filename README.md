# End-to-End DevOps Pipeline

## Overview
A production-ready DevOps project demonstrating containerization, cloud deployment, monitoring, and CI/CD automation for a two-tier application.

---

## Tech Stack
- Frontend: HTML, CSS  
- Backend: FastAPI  
- Containerization: Docker  
- Orchestration: Docker Compose  
- Monitoring: Prometheus, Grafana  
- CI/CD: GitHub Actions / Jenkins  
- Cloud: AWS / GCP / Azure  

---
